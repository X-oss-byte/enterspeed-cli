# es-cli view
Reference

Shortcut: `es-cli v`

Query views.

## Commands
|Command| |
|---|---|
|[es-cli view list](#es-cli-view-list)  |Queries views.   |

### es-cli view list

Query generated views.

```bash
es-cli view list <EnvironmentId> [--SourceId] 
                                 [--SchemaAlias]
                                 [--OriginId] 
```

#### Required Parameters
`EnvironmentId`

Id of environment to query.


#### Optional Parameters

`--SourceId`

`--SchemaAlias`

`--OriginId`
