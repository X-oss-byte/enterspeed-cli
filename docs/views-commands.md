### es-cli view list

Query generated views.

```bash
es-cli view list <environmentId> [--source-id] 
                                 [--schema-alias]
                                 [--origin-id] 
```

#### Required Parameters
`environmentId`

Id of environment to query.


#### Optional Parameters

`--source-id`

`--schema-alias`

`--origin-id`
